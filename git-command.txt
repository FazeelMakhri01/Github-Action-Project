git init                            to intitalize a git repo.
git status                          to check status
git add " filename"                 to move file from untracked -> staging
git commit -m "message"             to move file from staging -> tracked
git restore "filename"              to recover a file if it deletes after coming to tracking stage.
git remote add origin "url"         to setup your local git to push code to github.
git remote set-url origin "url"     to set the url with personal token.
